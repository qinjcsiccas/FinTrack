## 📝 记账核心协议 (核心规范)
看板通过解析你的 `Notes` 实现自动分类，请务必遵守 **“冒号协议”**：

### 1. 基本格式
**`大类：明细内容`** 或  **`大类1：明细；大类2：明细`**
* ✅ **正确示例**：`工资：3月薪水`、`旅游：去三亚`、`大件：Macbook`
* ✅ **多项并行**：`工资：绩效；理财：买入基金` (系统会自动抓取第一个有效大类进行统计)

### 2. 常用大类建议
看板具有自适应识别能力，建议统一使用以下关键词：
* **收入类**：`工资`、`收益`、`报销`、`红包`
* **支出类**：`房租`、`旅游`、`大件`、`日常`
* **资产转移**：`理财` (例如 `理财：买入余额宝`)。*注意：此类记录不计入收支盈亏，仅记录资产结构变化。*

## ⚙️ 核心功能模块

### 1. 📅 职业/生活里程碑 (侧边栏)
* **动态管理**：在侧边栏表格中直接点击日期列（弹出日历）或自由增减行。
* **自动分割**：资产趋势图会根据你定义的里程碑（如“入职XX公司”、“读博”）自动划分阶段。
* **速率计算**：系统会实时计算**当前阶段**及**近365日**的平均增长速率，反映你最新的“攒钱效率”。

### 2. 👁️ 隐私模式 (Privacy Mode)
* 勾选后，所有绝对金额（总资产、月盈亏、热力图数值）都会遮盖为 `****`。
* 适合在公共场合演示或截屏分享，同时保留趋势曲线和比例分布。

### 3. 📉 四大分析面板
* **趋势与月盈亏**：查看资产积累的“坡度”和每月净现金流。
* **进阶速率**：
    * **财富里程碑**：每攒够一个步长（默认10w）平均需要多少天。
    * **年度对比**：横向对比不同**自然年**的日均增长，直观感受赚钱能力的进化。
* **收支与分类**：解析 Notes 冒号前的关键词，自动生成消费/收入排行榜。
* **预测与热力图**：基于近一年速率，科学预测达成目标（如100w）的具体日期。

## 💡 常见问题 (FAQ)
* **Q: 为什么分类显示为“其他”？**
  - 请检查是否正确使用了中英文冒号（`:` 或 `：`）。
* **Q: 如何删除一个里程碑？**
  - 选中侧边栏表格中的那一行，按下键盘的 `Delete` 键即可。

---
**💡 提示**：本工具默认使用“近365日均积累”进行预测，这比全周期平均值更能反映你当下的财务战斗力。
